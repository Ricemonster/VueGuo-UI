# Row Attributes
| 参数        | 说明           | 类型  | 可选值 |  默认值 |
| ------------- |:-------------:| -----:|-----:|-----:|
| align     | 栅格排列的起始位置 | String  | left,center,right | ---
| gutter     | 栅格间隔 | Number,String  | --- | ---

# Col Attributes

| 参数        | 说明           | 类型  | 可选值 |  默认值 |
| ------------- |:-------------:| -----:|-----:|-----:|
| span      | 栅格占据的列数 | Number,String | 1~24   | 24 |
| offset      | 栅格左侧的间隔格数 | Number,String | ---   | --- |
