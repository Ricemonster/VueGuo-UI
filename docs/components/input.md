# Options

| 参数        | 说明           | 类型  | 可选值 |  默认值 |
| ------------- |:-------------:| -----:|-----:|-----:|
| value      | 输入框默认文字 | String | --- | --- |
| readonly   |   只读状态    |   Boolean | true,false | false |
| disabled | 禁止状态      |    Boolean | true,false | false|
